# AI Infra Learning Site

这是 `oneDuter` 的 AI 系统与 AI Infra 16 周学习驾驶舱。

- 在线网站：<https://oneDuter.github.io/ai-infra-learning-site/>
- AIInfra 上游课程：<https://github.com/Infrasys-AI/AIInfra>
- 个人 Fork：<https://github.com/oneDuter/AIInfra>

本仓库只包含静态网页文件，不包含书籍 PDF、私人笔记或私有仓库内容。

## 功能

- 16 周学习路线图
- “读、看、做、写、测”任务详情面板与完成闭环
- 每周书籍章节、AIInfra 个人 Fork 精确链接、实验和五问自测
- “读”任务按周打开站内 PDF 阅读器，每次只加载本周章节文件
- 学习进度统计与阶段筛选
- 深色/浅色主题
- 本地进度保存、JSON 导入和导出

网页由原生 HTML、CSS 和 JavaScript 构建，可直接通过 GitHub Pages 发布。
